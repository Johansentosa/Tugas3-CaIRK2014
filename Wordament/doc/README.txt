==============WORDAMENT SOLVER==============
Program ini adalah program untuk menampilkan semua jawaban 
kemungkinan kata-kata yang dapat ditemukan saat bermain Wordament.

Untuk menjalankannya, cukup merubah file "matriks.txt" yang terdapat pada folder bin
dengan soal pada game. Setiap sel, hanya dapat berisi 1 buah karakter.
Kemudian Program akan mengeluarkan daftar kata-kata beserta path untuk 
membentuk kata tersebut.
